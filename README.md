**##pneumonia_detection**  
Repositori ini berisikan model untuk mendeteksi pneumonia berdasarkan citra X-ray dada menggunakan CNN  

---

**## install Pustaka**  
pip install -r requirements.txt  

---

**## masuk ke direktori**  
cd pneumonia_detection  

---

**## running streamlit**  
streamlit run app.py
